## 📥 安裝說明

1. **在Android設備上安裝：**
   - 將APK文件傳輸到Android設備
   - 在設備上啟用"未知來源"應用安裝權限
   - 點擊APK文件進行安裝

2. **使用ADB安裝：**
   ```bash
   adb install 大學生小助手-v1.0-debug.apk
   ```

3. **如遇安裝問題，請參考：**
   - `APK_DEBUG_GUIDE.md` - 安裝故障排除指南
   - `APK_INSTALLATION_GUIDE.md` - 詳細安裝指南
